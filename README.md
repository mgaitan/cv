# CV / Resume

## Espanol

Este repositorio contiene mi CV generado con RenderCV a partir de YAMLs (ES/EN).

Como generar el PDF/HTML:

```bash
make
```

Salidas principales:
- `rendercv_output/Martín_E._Gaitán_CV_es.pdf`
- `rendercv_output/Martín_E._Gaitán_CV_en.pdf`
